# PRD: Foundation — Next.js + Tailwind v4 + App Shell

## Tasks

- [ ] Install dependencies. Run `npm install next@^19 react@^19 react-dom@^19 framer-motion@^11` and `npm install -D tailwindcss@^4 @tailwindcss/postcss@^4 postcss@^8 @types/react@^19 @types/react-dom@^19 @types/node@^22`, then update `package.json` scripts to add `"dev": "next dev"`, `"build": "next build"`, `"start": "next start"` while keeping existing test, lint, check-types scripts. `[test: npx tsc --noEmit]`
- [ ] Update TypeScript configuration. Modify `tsconfig.json`: set `module` to `esnext`, `moduleResolution` to `bundler`, `jsx` to `preserve`, `noEmit` to `true`, add `lib: ["dom","dom.iterable","esnext"]`, add `plugins: [{ "name": "next" }]`, add `paths: { "@/*": ["./*"] }`, set `include` to `[\"next-env.d.ts\",\"**/*.ts\",\"**/*.tsx\", \".next/types/**/*.ts\"]`, keep `exclude` as `[\"node_modules\",\".next\",\"dist\",\"out\"]`. `[test: npx tsc --noEmit]`
- [ ] Update gitignore. Append `.next/`, `out/`, and `next-env.d.ts` to the end of `.gitignore` file. `[test: npx biome check .gitignore]`
- [ ] Delete legacy entry point. Remove `src/index.ts` from the repository and verify no files import it. `[test: bash scripts/delete-src-index.sh]`
- [ ] Create Next.js configuration. Create `next.config.mjs` exporting default config with `output: 'export'`, `basePath: '/ralph-node-resume'`, `assetPrefix: '/ralph-node-resume'`, `images: { unoptimized: true }`, `trailingSlash: true`, and `reactStrictMode: true`. `[test: npx tsc --noEmit]`
- [ ] Create PostCSS configuration. Create `postcss.config.mjs` exporting `{ plugins: { '@tailwindcss/postcss': {} } }`. `[test: npx tsc --noEmit]`
- [ ] Configure public assets. Copy existing `profile.png` from repo root to `public/profile.png` (binary copy), create empty `public/.nojekyll` file to disable Jekyll processing on GitHub Pages. `[test: bash scripts/configure-public-assets.sh]`
- [ ] Create base path helper. Create `src/lib/basePath.ts` exporting constant `BASE_PATH = '/ralph-node-resume'` and function `withBasePath(path: string): string` that returns `${BASE_PATH}${path.startsWith('/') ? path : '/' + path}` for use in all JSX `src`/`href` attributes referencing static files under `public/`. `[test: npx jest tests/unit/create-basepath-helper.test.ts]`
- [ ] Create resume data module. Create `src/content/resume.ts` exporting typed objects: `profile` object with name, title, tagline, location, email, phone, linkedin, github, summary fields; `skills` array of `{ category, items }` for Frontend/AI/Infra/Backend categories; `experience` array of `{ company, location, role, period, bullets: { label, body }[], stack }` in reverse chronological order; `education` array of `{ title, detail, status? }`. All values must exactly match content from `resume.md`. `[test: npx jest tests/unit/transcribe-resume-data.test.ts]`
- [ ] Create global styles. Create `app/globals.css` with `@import "tailwindcss";`, a `@theme` block defining `--font-sans: "Inter", system-ui, sans-serif;` and neutral palette tokens (`--color-bg: #ffffff`, `--color-fg: #0a0a0a`, `--color-muted: #6b7280`, `--color-subtle: #f5f5f7`, `--color-border: #e5e5ea`), base element resets applying vars to html/body with smooth scroll behavior, and `.section-pad` utility class equivalent to `py-24 md:py-32 px-6 md:px-12 max-w-6xl mx-auto`. `[test: npx playwright test tests/e2e/setup-global-styles.spec.ts]`
- [ ] Create root layout. Create `app/layout.tsx` as server component exporting default `RootLayout({ children })` that renders `<html lang="en" className={inter.className}>` with `<body data-testid="app-body">` containing `<Nav />` and `{children}`, imports `./globals.css`, loads Inter font via `next/font/google` with `subsets: ['latin']` and applies its className to html, exports metadata with title `"Paul-Valentin Mini — Senior Software Developer"` and matching description. `[test: npx playwright test tests/e2e/create-root-layout.spec.ts]`
- [ ] Create navigation component. Create `src/components/Nav.tsx` as `'use client'` component rendering sticky top nav (`position: sticky; top: 0; z-index: 50`, white/blurred background via `backdrop-blur bg-white/70 border-b border-[var(--color-border)]`) with `data-testid="nav"`. Render "Paul-Valentin Mini" on left (`data-testid="nav-brand"`), anchor links for `#profile`, `#skills`, `#experience`, `#education` (`data-testid="nav-link-profile"`, `nav-link-skills`, `nav-link-experience`, `nav-link-education`) that smooth-scroll via browser default behavior. Mobile-first: links collapse to hamburger button (`data-testid="nav-toggle"`) below md breakpoint toggling dropdown panel (`data-testid="nav-menu"`). `[test: npx playwright test tests/e2e/create-nav-component.spec.ts]`
- [ ] Create section wrapper component. Create `src/components/Section.tsx` as reusable component accepting `{ id: string, title?: string, testId: string, children: React.ReactNode }`, rendering `<section id={id} data-testid={testId}>` with `.section-pad` class; if title provided render `<h2 data-testid={`${testId}-title`}>` using large Apple-style typography (`text-4xl md:text-6xl font-semibold tracking-tight mb-12`). `[test: npx playwright test tests/e2e/create-section-component.spec.ts]`
- [ ] Create home page. Create `app/page.tsx` as server component rendering `<main data-testid="home">` with placeholder sibling `<Section>` stubs for `#hero`, `#profile`, `#skills`, `#experience`, `#education` (each using Section with empty children and distinct testIds: `section-hero`, `section-profile`, `section-skills`, `section-experience`, `section-education`) ready for Ticket 2 to replace children. `[test: npx playwright test tests/e2e/create-home-page.spec.ts]`
