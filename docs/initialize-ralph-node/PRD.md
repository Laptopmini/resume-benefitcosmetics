# PRD: Project Infrastructure & Tooling Setup

## Tasks

- [ ] Configure TypeScript. Create a `tsconfig.json` with `target` set to `ES2022`, `module` set to `NodeNext`, `moduleResolution` set to `NodeNext`, `strict` enabled, `esesModuleInterop` enabled, `skipLibCheck` enabled, `forceConsistentCasingInFileNames` enabled, `outDir` set to `dist/` and `rootDir` to `src/`. Also have the config `include` any `src/**/*` files and create a file `src/index.ts` with only `export {};` and a comment indicating its a placeholder as the project is empty. Have the `types` array includes `jest` and `node`. Make sure to `exclude` the folders `.next`, `dist` and `node_modules`. `[test: npm run check-types]`
- [ ] Configure Jest. Create a `jest.config.js` using the `@swc/jest` preset for a standard Node/TypeScript environment. Add a `"test:unit"` script to `package.json` that runs Jest using the option `--passWithNoTests`. Add both `*.spec.ts` and `*.test.ts` as a pattern to discover test files. `[test: npm run test:unit]`
- [ ] Configure Playwright. Create a `playwright.config.ts` set to run headless Chromium, with `baseURL` set to `http://localhost:3000` as a placeholder and `timeout` set to `10000`. Add a `"test:e2e"` script to `package.json` that runs Playwright. Create a sanity test at `tests/e2e/setup.spec.ts` that navigates to `about:blank` and asserts the page title is an empty string. Add both `*.spec.ts` and `*.test.ts` as a pattern to discover test files. `[test: npm run test:e2e]`
- [ ] Wire up a root test script. Update the `"test"` script in `package.json` to run `test:unit` and `test:e2e` sequentially. Verify it exits cleanly. `[test: npm test]`
